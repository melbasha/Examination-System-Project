# Online Exam System

## Overview

The Online Exam System is a simple web-based application designed to help students take exams online in an easy and organized way. The system provides a user-friendly interface that allows users to answer questions and view their results after completing the exam.

## Features

* Simple and clean user interface
* Multiple choice questions
* Automatic score calculation
* Instant result display after finishing the exam
* Easy navigation between questions

## Technologies Used

This project was built using basic web technologies:

* **HTML** for the structure of the pages
* **CSS** for styling and layout
* **JavaScript** for exam logic and score calculation

## How It Works

1. The user opens the exam page.
2. The exam questions are displayed on the screen.
3. The student selects the correct answers.
4. After submitting the exam, the system calculates the score automatically.
5. The final result is displayed to the user.

## Purpose of the Project

This project was created as a practice project to demonstrate the use of front-end web technologies in building an interactive exam system.
